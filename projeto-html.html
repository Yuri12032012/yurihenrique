Projeto HTML

Este é um projeto básico desenvolvido com HTML5 para praticar a estruturação de páginas web.

Conteúdos utilizados

  • Títulos e parágrafos
  • Listas ordenadas e não ordenadas
  • Links
  • Estrutura semântica com HTML5

Objetivo

O objetivo deste projeto foi aprender os fundamentos da construção de páginas web utilizando HTML puro.
